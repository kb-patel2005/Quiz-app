# Quiz Leaderboard Project

## 🚀 Overview
This project is a Firebase‑powered quiz app with a leaderboard.  
After completing a quiz, users enter their name, and their score is saved in Firestore.  
The leaderboard displays the **Top 10 scores** for each quiz, ordered by score and completion time.
